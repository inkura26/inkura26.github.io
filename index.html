核心功能

为极致阅读体验设计

从网络书源到本地书籍，从精准排版到 iCloud 同步， 墨读的每一项功能都经过深度打磨。

📖

精准 TextKit 排版

行级分页算法替代逐字形循环，排版性能提升 10–50 倍。支持 TextKit 2（iOS 16+）自动升级，首行缩进、行距、字距精细可调。

🌐

工业级书源引擎

完整支持 XPath、CSS 选择器、JSONPath 和 JavaScript 规则四种解析方式。完全兼容 Android 版书源格式，JSON 直接导入。

☁️

iCloud 全端同步

书架、阅读进度、书源配置、阅读设置、书签——跨 iPhone / Mac 实时双向同步。支持 backup.zip 一键导入。

📦

多格式本地导入

EPUB 3（ZIPFoundation 原生解压）、TXT（GBK/UTF-8/Big5 自动检测）、PDF 阅读。大文件流式读取，消除 OOM 风险。

🔒

原子性存储引擎

RobustStorageManager 基于 Core Data 实现 CRC32 校验、原子写入、LRU 备份三重保护，数据永不丢失。

🔊

TTS 语音朗读

系统原生 AVSpeechSynthesizer，支持多语音选择、语速 / 音调 / 音量调节，后台朗读不中断。

🎨

五款内置主题

米纸、夜间、护眼绿、海洋蓝、极夜。支持自定义背景色 / 字色，跟随系统深色模式自动切换。

⚡

高容错网络请求

AsyncSemaphore 并发控制（上限 8 路），指数退避重试，LRU 内存缓存，编码自动检测（UTF-8 / GBK / Big5）。

🧹

智能正文净化

内置 10 条广告过滤规则，正确解码 HTML 数字实体（{ / {），支持用户自定义正则替换规则。

书源引擎

完整兼容
3.0 书源生态

墨读的书源引擎参照 Legado 开源协议设计（Clean Room）， 用户可直接导入数千个社区维护的书源，无需重新配置。

📐

XPath 引擎（完整 // 后代语法）

修复 isDescendant 逻辑错误，正确识别 //div/span 等后代选择路径，解决 30% 旧版书源失效问题。

🔧

CSS / JSONPath / 正则三重降级

XPath 解析失败自动降级到 SwiftSoup CSS 选择器，确保所有书源均可兼容解析。

🔑

CryptoJS 内置兼容层

内置 MD5 / SHA256 / AES / Base64 支持，使用 CryptoKit 现代 API，无废弃接口，App Store 审核通过。

🌍

java.ajax / java.connect 同步桥接

完整模拟 3.0 Android 的 java 对象接口，书源 JS 脚本无需修改即可运行。

book_source.json
{
"bookSourceName": "图书馆",
"bookSourceUrl": "https://x.com",
"searchUrl": "...?q={{key}}",
"ruleSearch": {

},
